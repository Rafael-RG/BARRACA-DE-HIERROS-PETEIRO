# Imágenes para Banner

Esta carpeta contiene las imágenes utilizadas en los banners de la aplicación.

## Imágenes requeridas:

### 1. `banner-hierros.jpg`
- **Uso:** Banner principal en la página de inicio
- **Descripción:** Imagen de materiales de construcción, hierros, chapas o ambiente de barraca
- **Tamaño recomendado:** 1920x600px (proporción 16:5)
- **Formato:** JPG o WebP
- **Sugerencias de búsqueda:**
  - "steel construction materials"
  - "metal sheets warehouse"
  - "construction steel beams"
  - "barraca de hierros"

### 2. `sobre-nosotros.jpg`
- **Uso:** Imagen de fondo en la sección "Sobre Nosotros"
- **Descripción:** Imagen que represente trabajo, construcción, equipo o instalaciones
- **Tamaño recomendado:** 1920x600px (proporción 16:5)
- **Formato:** JPG o WebP
- **Sugerencias de búsqueda:**
  - "construction team working"
  - "warehouse facility"
  - "steel workshop"
  - "construcción trabajo en equipo"

## Sitios recomendados para imágenes gratuitas:
- **Unsplash** (unsplash.com)
- **Pexels** (pexels.com)
- **Pixabay** (pixabay.com)

## Notas:
- Las imágenes se muestran con una capa de opacidad (30% banner principal, 30% sobre nosotros)
- El tamaño del archivo debe ser optimizado para web (preferiblemente < 500KB)
- Asegúrate de tener los derechos de uso de las imágenes
